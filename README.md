# SQL Injection Protection System

## Project Overview
This project was developed as part of a Cloud Computing and Cyber Security internship project.

The system protects user data from SQL injection attacks using parameterized queries, AES-256 encryption, capability code authentication, and double-layer security mechanisms.

## Features
- SQL Injection Prevention
- AES-256 Encryption
- Secure User Authentication
- Capability Code Verification
- Double-Layer Security
- SQLite Database Integration
- Secure Credential Storage

## Technologies Used
- Python
- SQLite
- Cryptography Library
- AES Encryption

## How To Run

```bash
python app.py
```
